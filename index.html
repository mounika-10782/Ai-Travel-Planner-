How the AI Works
What happens when you click Generate
1
Your form data is sent to Python (Flask) via POST request
2
Python builds a Goal Stack — a list of goals to achieve
3
Each goal is resolved using a STRIPS Operator (TRAVEL, BOOK-HOTEL, VISIT, MEALS)
4
Python checks the Knowledge Base for distances, costs, attractions
5
The final plan is sent back and shown as a web page
